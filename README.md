# Nutriapp
Nutrtional app with IA
NutriAI – Fotografia e Planejamento de Dieta



NutriAI é um aplicativo de inteligência artificial que permite aos usuários fotografar seus pratos e receber informações nutricionais completas (calorias, proteínas, carboidratos, gorduras, fibras e micronutrientes). Além disso, o app sugere dietas personalizadas e oferece recursos premium através de assinatura.


---

Funcionalidades

Reconhecimento de alimentos por foto:
Identificação automática de alimentos e cálculo nutricional.

Sugestões de dietas e refeições:
Planejamento diário ou semanal com base nos objetivos do usuário (emagrecimento, ganho muscular, manutenção de peso, saúde geral).

Perfil e histórico:
Registro de peso, altura, idade e objetivos. Histórico de refeições com gráficos de progresso.

Monetização via assinatura:
Teste gratuito de 7 dias, com recursos premium como receitas exclusivas e relatórios detalhados.

Interface visualmente atraente:
Layout moderno, cores saudáveis, imagens de comida de fundo e cards interativos.



---

Tecnologias

Google Studio IA – para desenvolvimento visual e lógica de app.

API Gemini – reconhecimento de alimentos em imagens.

JavaScript – lógica do app e integração de funções.

Stripe / PayPal – pagamentos para assinaturas.

Banco de dados interno – armazenamento de perfil e histórico nutricional.
